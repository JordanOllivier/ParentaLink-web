# Screenshots

Place your iOS / Android screenshots here. Used by `index-new.html`.

## Naming convention

The site references these filenames — keep them as-is, or tell Claude to rename:

- `calendar.png` — Calendar / week view
- `expenses.png` — Expenses list with balance
- `documents.png` — Documents folder view
- `rules.png` — Rules screen (optional)
- `home.png` — Home dashboard (optional)

## Recommended specs

- **Format**: PNG (transparent background OK)
- **Source**: iPhone 6.5" (1284×2778) or 5.5" (1242×2208), or Android equivalent
- **No device frame**: pure UI, the site adds the phone frame in CSS
- **Light mode preferred** for the hero; can mix with one dark-mode shot for variety

## How to capture

### iOS (Xcode Simulator)
1. Run on iPhone 15 Pro simulator
2. `Cmd+S` to save screenshot to Desktop
3. Crop to UI area only (no status bar artifacts)

### iOS (real device)
1. Side button + Volume Up
2. Airdrop / iCloud to Mac

### Android (Android Studio Emulator)
1. Camera icon in emulator toolbar
2. Captures to `~/Documents/AndroidStudio/captures/` by default

### Android (real device)
1. Power + Volume Down
2. ADB pull or Google Drive
